# github@[vimrc](https://github.com/amix/vimrc)

# 安装

``` bash
# 基础版
git clone git://github.com/amix/vimrc.git ~/.vim_runtime
sh ~/.vim_runtime/install_basic_vimrc.sh

# 炫酷版
git clone git://github.com/amix/vimrc.git ~/.vim_runtime
sh ~/.vim_runtime/install_awesome_vimrc.sh
```

# 卸载

1. 删除 ~/.vim_runtime
2. 删除 ~/.vimrc 中有关的配置
